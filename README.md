# 🎓 Real-Time Student Presence Portal

## 🔹 Description

The Real-Time Student Presence Portal is a full-stack web application designed to track and manage student attendance in real time. It provides an interactive user interface for students and admins, along with a robust backend system for data processing and storage.

##  Technologies Used

* Frontend: React.js, HTML, CSS
* Backend: Java (Spring Boot)
* Database: MySQL
* API Testing: Postman

##  Features

* Student Registration & Login
*  Real-Time Attendance Tracking
*  View Attendance Records
*  Responsive UI using HTML & CSS
*  Dynamic frontend using React.js
*  Secure backend using Spring Boot
*  REST API communication

---

##  System Architecture

* React.js frontend built using HTML & CSS
* Frontend communicates with backend via REST APIs
* Spring Boot backend handles logic and database operations
* MySQL database stores student and attendance data

---

## \ How to Run the Project

### Backend (Spring Boot)

1. Open project in Eclipse / IntelliJ
2. Configure MySQL database
3. Run the Spring Boot application

###  Frontend (React)

1. Go to frontend folder
2. Run `npm install`
3. Run `npm start`
